Erklärung

Ziwschen zwei form-Tags mit den Attributen action="zielseite" und method="get oder post" (wären wichtig, wenn wir die Daten weiterverarbeiten wollten) werden input-Elemente angelegt mit folgenden Attributen:

type
password, text, checkbox, radio, number
name
Der Name der Variable, wenn wir die eingegeben Daten weiterverarbeiten würden, radio-checkboxes mit dem selben Namen können nicht gleichzeitig ausgewählt werden
value
Wert, der bei Auswahl dieser Checkbox/Listenoption/etc übermittelt wird. Wird dieses Attribut bei einem Textfeld spezifiziert, wird der Wert autmatisch eingetragen.
placeholder
Wird im Textfeld angezeigt bis der Benutzer etwas eingibt
checked
Markiert automatisch eine Checkbox

Label

Mit labeln kann man die Nutzung einer Webseite mit der Maus verbessern, denn der Nutzer kann auf den Text des Labels klicken und dadurch die Checkbox oder das Textfeld aktivieren. Diesen Effekt erreicht man, indem man das input-Element in ein Label-Element steckt:
<label><input type="radio" name="gender" >Male</label>
oder indem man das input-Tag mit einem ID-Attribut versieht und das Label mit einem for-Attribut desselben Wertes ausstattet:

<label for="username">Username</label>
<input type="text" name="username" id="username">

Fieldset

Mit fieldset und legend kann man mehrere Inputs zusammenfassen und betiteln.

Listen

Mit select und mehreren option erstellt man eine Liste aus der der User etwas auswählen kann. Die zu Beginn ausgewählte Stelle bestimmt man mit dem Attribut selected im option-Tag.

Text Area

Ein größeres Textfeld erstellt man mit dem textarea Element und den Attributen rows und cols.

Absenden

Mit einem button Element mit dem Attribut type="submit" kann man das formular abschicken.
